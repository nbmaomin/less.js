## 2. 运行

### 2.1. Node 版本
Koa2 使用了 async/await 等新语法，请保证 Node 版本在 7.6 及以上。

### 2.2. 命令
```bash
# 安装依赖
$ npm install

# JS 代码校验
$ npm run eslintfix
$ npm run eslint

# 启动
$ npm start
```
